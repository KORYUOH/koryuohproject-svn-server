Visual Leak Ditector Using Manual

1. 必要なファイルをプロジェクトにコピー
VLD がインストールされたディレクトリ（標準では C:\Program files\Visual Leak Detector）から、bin, include, lib の 3 つのディレクトリをまるごと、C++ プロジェクトのディレクトリ以下の適当な位置にコピーしましょう。（例：$(ProjectDir)\vendor\vld-1.9d\）
 
2. C++ プロジェクトのプロパティを設定
VS2005 で C++ プロジェクトを開き、プロジェクトのプロパティのダイアログを開きます。
ちなみに、VLD はリリースビルド時にはリンクされませんので、プロパティページはデバッグビルド用のページを開いておいてください。
 
[構成プロパティ] - [C/C++] - [全般] のページの、[追加のインクルード ディレクトリ] の項目に
$(ProjectDir)\vendor\vld-1.9d\include
を追加します。
 
[構成プロパティ] - [C/C++] - [コード生成] のページの、[ランタイムライブラリ] の項目は、
マルチスレッド デバッグ（/MTd）
にしておきます。
 
[構成プロパティ] - [リンカ] - [全般] のページの、[追加のライブラリ ディレクトリ] の項目に
$(ProjectDir)\vendor\vld-1.9d\lib
を追加します。
 
[構成プロパティ] - [リンカ] - [入力] のページの、[追加の依存ファイル] の項目に
$(ProjectDir)\vendor\vld-1.9d\lib\vld.lib
を追加します。
 
[構成プロパティ] - [ビルドイベント] - [ビルド後のイベント] に
copy $(ProjectDir)\vendor\vld-1.9d\bin $(OutDir)
を追加します。
 
3. ソースに #include <vld.h> を一か所だけ入れる
main() 関数のある .cpp ファイルの先頭に、
#include <vld.h>
という一行を追加します。
